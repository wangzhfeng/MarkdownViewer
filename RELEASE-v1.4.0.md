# MarkdownViewer v1.4.0 - 大纲视图和快速跳转

**发布日期：** 2026-03-10  
**版本类型：** 功能更新

---

## ✨ 新功能

### 📑 大纲视图

新增右侧大纲面板，显示文档的标题层级结构，帮助你快速了解文档内容。

**打开方式：**
- 点击右上角的 **📑 大纲** 按钮
- 按键盘上的 **O** 键

**功能特点：**
- 自动识别 H1-H6 标题
- 点击大纲项快速跳转到对应位置
- 滚动时自动高亮当前阅读的章节
- 平滑滚动动画
- 支持随时关闭（再次按 O 键或点击 ✕ 关闭）

### ⚡ 快速跳转

**快捷键：**
- `O` - 切换大纲视图
- `1` - 跳转到第一个 H1 标题
- `2` - 跳转到第一个 H2 标题
- `3` - 跳转到第一个 H3 标题
- `4` - 跳转到第一个 H4 标题
- `5` - 跳转到第一个 H5 标题
- `6` - 跳转到第一个 H6 标题

---

## 🛠️ 技术实现

### 前端部分

**CSS 新增：**
- `.outline-panel` - 大纲面板样式（固定右侧，250px 宽）
- `.outline-tree` - 大纲树形结构
- `.outline-toggle` - 切换按钮
- `.with-outline` - 正文边距调整

**JavaScript 新增：**
- `generateOutline()` - 自动扫描标题生成大纲
- `scrollToHeading()` - 平滑滚动到目标标题
- `highlightActiveHeading()` - 高亮当前章节
- `onScroll()` - 监听滚动自动更新高亮
- `toggleOutline()` - 切换大纲显示

### 后端部分

**C# 修改：**
- `ViewerControl.cs` - 添加 `KeyboardCallback` 类，处理键盘事件
- `InjectKeyboardHandler()` - 添加 O 键监听
- `OnKeyPressed()` - 处理 O 键和数字键

---

## 📋 使用说明

1. **打开 Markdown 文件** - 在 Total Commander 中预览 .md 文件
2. **查看大纲** - 按 `O` 键或点击右上角按钮
3. **快速导航** - 点击大纲项或按数字键跳转
4. **关闭大纲** - 再次按 `O` 键或点击 ✕ 关闭

---

## 🎯 适用场景

- 📖 阅读长文档（技术文档、教程、书籍）
- 🔍 快速定位特定章节
- 📝 编写结构化文章时查看整体框架

## ⚠️ 注意事项

- 大纲视图仅识别 H1-H6 标题
- 快捷键在预览模式下生效
- 大纲面板会占用右侧 250px 空间
